### To run

- Have node installed. Everything above 20 should work (worked on 20.18.1)

- Clone the project:

```
git clone https://github.com/vjekooo/super-duper-palm-tree
```

- Install the dependencies:

```
yarn install
```

- Run development server:

```
yarn dev
```

Wait for the browser to open

### To test

```
yarn test
```
